# GreenTech MVP Sostenible

## Descripción del proyecto
GreenTech MVP Sostenible es una iniciativa enfocada en auditar y refactorizar una **landing page** para concienciar sobre el cambio climático.  
El proyecto transforma una web ineficiente en una solución **ligera, rápida y sostenible**.


## Impacto ambiental
Reducir el peso de una web implica:

- Menor consumo energético en **servidores**  
- Menor transferencia de **datos en la red**  
- Menor uso de recursos en el **dispositivo del usuario**  


## Economía circular y sostenibilidad
Este proyecto fomenta:

- Alargar la **vida útil de los dispositivos**  
- Reducir la **obsolescencia programada**  
- Disminuir la generación de **residuos electrónicos (e-waste)**
