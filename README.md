# GreenTech MVP Sostenible

## Descripción del proyecto
GreenTech MVP Sostenible es una iniciativa enfocada en auditar y refactorizar una **landing page** para concienciar sobre el cambio climático.  
El proyecto transforma una web ineficiente en una solución **ligera, rápida y sostenible**.

---

## Problema detectado
El código original presentaba múltiples ineficiencias:

- Uso de **Bootstrap** para estilos simples  
- Uso innecesario de **jQuery**  
- Uso de **Moment.js** únicamente para obtener el año  
- Uso de **Font Awesome** para un solo icono  

Esto provocaba:

- Alto consumo de datos  
- Mayor uso de CPU  
- Tiempos de carga elevados  
- Impacto negativo en dispositivos de gama baja  

---

## Solución aplicada
Se realizó una refactorización completa:

- Eliminación de librerías innecesarias  
- Uso de **HTML, CSS y JavaScript nativo**  
- Separación de HTML y CSS para mejor mantenibilidad  
- Optimización de la imagen de fondo (reducción de resolución)  
- Simplificación del código  

---

## Resultados obtenidos
- Reducción de más de **400KB** en recursos  
- Mejora significativa del **tiempo de carga**  
- Disminución del **consumo energético**  
- Código más **limpio, mantenible y escalable**  

---

## Impacto ambiental
Reducir el peso de una web implica:

- Menor consumo energético en **servidores**  
- Menor transferencia de **datos en la red**  
- Menor uso de recursos en el **dispositivo del usuario**  

Esto contribuye directamente a reducir la **huella de carbono digital**.

---

## Economía circular y sostenibilidad
Este proyecto fomenta:

- Alargar la **vida útil de los dispositivos**  
- Reducir la **obsolescencia programada**  
- Disminuir la generación de **residuos electrónicos (e-waste)**

El mejor software no es el más complejo, sino el más **eficiente**.  
Reducir la “grasa digital” es una estrategia clave para desarrollar aplicaciones **sostenibles** y responsables con el **medio ambiente**.
